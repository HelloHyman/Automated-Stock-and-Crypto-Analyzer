========================================
  Stock Technical Analyzer
  Quick Start Guide
========================================

FOR USERS:
----------
1. Double-click StockAnalyzer.exe
2. Enter a stock symbol (e.g., AAPL, MSFT, TSLA)
3. Click "Analyze Stock"
4. Customize chart settings and enjoy!

No Python or installation required!


FOR DEVELOPERS (Building):
--------------------------
1. Open Command Prompt in the project root folder
2. Run: BUILD.bat
   (or: WindowsBuild\build_windows.bat)
3. Wait for build to complete
4. Find executable in WindowsBuild\dist\StockAnalyzer.exe

Requirements:
- Python 3.8+
- Internet connection

See README_BUILD_AND_INSTALL.md for detailed instructions.
